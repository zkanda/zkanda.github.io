Recently upgraded Windows on my dual boot laptop causes problems on my bootloader.

So I upgraded my Windows software to the latest insider preview and it kind of messed up my laptop.

Here’s the error message:

EFI stub: ERROR: failed to read file.
Trying to load files to higher address
EFI stub: ERROR: failed to read file.

Here’s the step I did to resolved it.

  1. Boot in an ArchLinux liveusb.
  2. Mount your root and boot partition.
  3. Chroot into it.
  4. And reinstall the bootloader.
mkinitcpio -p linux
bootctl install

Reboot and it should be good now.
